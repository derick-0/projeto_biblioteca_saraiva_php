# projeto_biblioteca_saraiva_php

O projeto **biblioteca_saraiva_php** é uma aplicação desenvolvida em PHP com o objetivo de gerenciar e organizar os livros de uma biblioteca virtual. Ele permite a adição, atualização, exclusão e consulta de livros, facilitando a gestão do acervo de uma biblioteca digital.

A aplicação foi projetada para ser simples e fácil de usar, com uma interface amigável e uma estrutura de código limpa e escalável.

Funcionalidades

- Cadastro de Livros: Permite adicionar novos livros à biblioteca com informações como título, autor, categoria e ano de publicação.
- Listagem de Livros: Exibe uma lista de todos os livros cadastrados, com a possibilidade de ordenar por diferentes critérios.
- Edição de Livros: Permite editar as informações dos livros cadastrados, como título, autor, e categoria.
- Exclusão de Livros: Permite excluir livros da biblioteca.
- Busca por Livros: Oferece uma funcionalidade de busca para encontrar livros com base em palavras-chave (título, autor ou categoria).
- Interface Simples: A aplicação possui uma interface simples e intuitiva, com navegação fácil e clara.

-linguagens Utilizadas

- PHP: Linguagem de programação utilizada para o desenvolvimento do backend da aplicação.
- MySQL: Sistema de banco de dados utilizado para armazenar os dados dos livros.
- HTML/CSS: Utilizados para a criação da interface de usuário.
- JavaScript: Para algumas interações dinâmicas na interface do usuário.

- Requisitos

- Servidor PHP com suporte a MySQL.
- Banco de dados configurado com as tabelas necessárias (ver seção **Instalação**) .
- Navegador moderno para acessar a interface.
